

# 开发环境及要求

?> 本章节将阐述使用ViewerSDK的开发场景以及硬件需求。



## 系统环境

iOS的打包需要使用`Xcode`，`Xcode`是`OSX`独占的应用。



## 软件要求

| 名称    | 版本                          |
| ------- | ----------------------------- |
| JDK     | JDK 1.7.0+                    |
| ADT     | Android 7.0(API Level 24)+    |
| Unity   | 2018.1.0+ (推荐使用2018.2.0+) |
| Vuforia | 7.2.23+                       |
